Projetos

Signalist

Aplicação Full Stack de análise de investimentos com IA.

Acessar Projeto →
O objetivo do projeto foi criar uma experiência de onboarding personalizada, onde o sistema analisa o perfil do usuário e gera relatórios exclusivos automaticamente. Combina dados de mercado em tempo real com Inteligência Artificial Generativa.
Desafios & Soluções
  • • Event-Driven: Inngest para processamento assíncrono e filas, evitando lentidão na interface.
  • • AI Contextual: Google Gemini para processar dados e gerar e-mails contextuais.
  • • Segurança: Better-Auth com Cookies Seguros (__Secure) e Middleware no Next.js.
Tech Stack
Next.js 14 TypeScript Tailwind CSS Shadcn UI MongoDB Server Actions Vercel Inngest
Decisão de Engenharia:

O desenvolvimento priorizou a arquitetura Desktop-First, focando na complexidade do backend e visualização de dados.
